# v1.0 — 旧书卷风格重构

> 日期：2026-05-12
> 范围：`web/static/index.html`, `web/panels.py`, `web/post_lesson.py`, `web/server.py`

## 计划

### 目标
1. **配色重构**：暗夜星空主题 → 旧书卷/羊皮纸暖色方案
2. **布局重构**：右侧面板 → 左侧竖排菜单 + 抽屉面板 + 右侧对话区
3. **Bug 修复**：4 个已知问题
4. **新功能**：未读消息标记已读（微信群聊归档）

### 设计决策
- 配色方向：旧书卷/羊皮纸风（用户选择方案3）
- 菜单形态：竖向标签切换（用户选择方案1）
- 面板交互：桌面端抽屉滑入（与菜单同侧），移动端底部滑出

纸纹理：`body::before` 横向纤维纹 + `body::after` SVG 噪点叠加 `animation: grain`。

### 2. 布局结构

```
旧布局：main > [chat-panel | right-panel(340px, 右侧标签)]
新布局：main > [left-menu(160px) | panel-content-wrapper(0→340px) | chat-panel(flex:1)]
```

- 面板抽屉：`panel-content-wrapper` 宽度过渡 `0.3s cubic-bezier`
- 关闭时 `width: 0; overflow: hidden`，不占用空间
- 菜单标签：`border-left` 金色指示线，active 态渐变背景
- 移动端：菜单变横向 tab bar，面板从底部 `translateY` 滑入

### 3. Markdown 渲染增强 (`panels.py`)

新增函数：
- `_convert_tables()` — `|...|` 格式 → `<table>`，检测分隔行 `|:---|`
- `_is_table_row()` / `_is_table_separator()` — 表格行检测
- `_build_table()` — 首行 `<th>`、后续 `<td>`

内部独白处理：
- `*(...)*` → `<span class="inner-thought">（...）</span>`（先于斜体）
- 斜体正则收窄为 `*(\w+)*`，避免误匹配

修复：span 标签转义中 `&quot;` → `"`，与 `< → &lt;` 后的实际字符匹配。

### 5. 标记已读功能

新增后端：
- `POST /api/panels/wechat/archive` (`server.py`)
- `archive_unread_to_group()` (`panels.py`) — 追加未读到 `wechat_group.md`（带时间戳），清空 `wechat_unread.md`，清除面板缓存

新增前端：
- `_injectPanelActions('wechat')` — 在微信面板顶部注入按钮
- `archiveWechatUnread()` — 调 API，成功后 `delete panelCache['wechat']` 并重新加载
